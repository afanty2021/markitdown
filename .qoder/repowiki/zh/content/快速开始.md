## 简介

MarkItDown 是一个轻量级的 Python 工具，用于将各种文件格式转换为 Markdown，特别适用于 LLM（大语言模型）和文本分析管道。它支持从 PDF、Word 文档、Excel 表格、PowerPoint 演示文稿、图像、音频等多种文件格式中提取内容并转换为结构化的 Markdown 格式。

### 支持的文件格式

- **文档格式**: PDF、Word (.docx)、Excel (.xlsx/.xls)、PowerPoint (.pptx)
- **图像格式**: JPEG、PNG、GIF 等（包含 EXIF 元数据和 OCR）
- **音频格式**: WAV、MP3（包含语音转录）
- **网页格式**: HTML、RSS、Wikipedia 页面
- **其他格式**: ZIP 文件、EPUB、Jupyter Notebook (.ipynb)、Outlook 邮件 (.msg)
- **在线内容**: YouTube 视频、Bing 搜索结果

## 安装要求

### 系统要求
- Python 3.10 或更高版本
- 推荐使用虚拟环境以避免依赖冲突

### 可选依赖项

MarkItDown 提供了灵活的可选依赖项系统，您可以根据需要选择性安装：

| 组合 | 描述 | 命令 |
|------|------|------|
| `[all]` | 安装所有可选依赖项 | `pip install 'markitdown[all]'` |
| `[pdf]` | PDF 文件支持 | `pip install 'markitdown[pdf]'` |
| `[docx]` | Word 文档支持 | `pip install 'markitdown[docx]'` |
| `[pptx]` | PowerPoint 支持 | `pip install 'markitdown[pptx]'` |
| `[xlsx]` | Excel 支持 | `pip install 'markitdown[xlsx]'` |
| `[audio-transcription]` | 音频转录支持 | `pip install 'markitdown[audio-transcription]'` |
| `[youtube-transcription]` | YouTube 转录支持 | `pip install 'markitdown[youtube-transcription]'` |
| `[az-doc-intel]` | Azure 文档智能支持 | `pip install 'markitdown[az-doc-intel]'` |

## 命令行使用指南

### 基本语法

```bash
markitdown [选项] <文件路径或URL>
```

### 常用命令示例

#### 1. 将文件转换为 Markdown 并输出到控制台

```bash
# 基本用法
markitdown example.pdf

# 处理 Word 文档
markitdown document.docx

# 处理 Excel 表格
markitdown spreadsheet.xlsx

# 处理 PowerPoint 演示文稿
markitdown presentation.pptx
```

#### 2. 将输出重定向到文件

```bash
# 使用 > 运算符
markitdown example.pdf > output.md

# 使用 -o 参数指定输出文件
markitdown example.pdf -o output.md

# 处理多个文件
for file in *.docx; do
    markitdown "$file" -o "${file%.docx}.md"
done
```

#### 3. 从标准输入读取内容

```bash
# 通过管道读取
cat example.pdf | markitdown

# 使用重定向
markitdown < example.pdf

# 处理压缩文件
gzip -dc archive.zip | markitdown -o archive.md
```

#### 4. 处理远程 URL

```bash
# 转换网页内容
markitdown https://example.com/page

# 转换 Wikipedia 页面
markitdown https://en.wikipedia.org/wiki/Markdown

# 转换 Bing 搜索结果
markitdown https://www.bing.com/search?q=python+tutorial
```

### 高级命令行选项

#### 1. 提供文件类型提示

### 错误处理

```python

md = MarkItDown()

try:
    result = md.convert("unsupported_format.xyz")
    print(result.markdown)
except UnsupportedFormatException:
    print("不支持的文件格式")
except FileConversionException as e:
    print(f"转换失败: {e}")
    for attempt in e.attempts:
        print(f"尝试 {attempt.converter.__class__.__name__}: {attempt.exc_info[1]}")
```

## 常见使用场景

### 场景 1：批量转换文档

```bash
#!/bin/bash
# 批量转换脚本

INPUT_DIR="documents"
OUTPUT_DIR="markdown"

mkdir -p "$OUTPUT_DIR"

for file in "$INPUT_DIR"/*; do
    if [ -f "$file" ]; then
        filename=$(basename "$file")
        output="${OUTPUT_DIR}/${filename%.*}.md"
        
        echo "转换 $filename..."
        markitdown "$file" -o "$output"
        
        # 检查转换是否成功
        if [ $? -eq 0 ]; then
            echo "✓ 成功: $output"
        else
            echo "✗ 失败: $filename"
        fi
    fi
done
```

### 场景 2：从数据库导出内容

```python
import sqlite3

# 查询文档内容

for row in cursor.fetchall():

    # 创建临时文件
    temp_file = f"/tmp/doc_{doc_id}.pdf"
    with open(temp_file, 'wb') as f:
        f.write(content)
    
    # 转换为 Markdown

    # 处理转换结果...
    print(f"文档 {filename} 的内容长度: {len(result.text_content)} 字符")
    
    # 清理临时文件

    os.unlink(temp_file)

```

### 场景 3：处理用户上传的文件

```python
from flask import Flask, request
from markitdown import MarkItDown
import tempfile
import os

app = Flask(__name__)
md = MarkItDown()

@app.route('/upload', methods=['POST'])
def upload_file():
    if 'file' not in request.files:
        return "未找到文件", 400
    
    file = request.files['file']
    if file.filename == '':
        return "未选择文件", 400
    
    # 保存上传的文件
    with tempfile.NamedTemporaryFile(delete=False) as temp:
        file.save(temp.name)
        temp_filename = temp.name
    
    try:
        # 转换文件
        result = md.convert(temp_filename)
        
        # 返回 Markdown 内容
        return {
            'success': True,
            'content': result.markdown,
            'metadata': result.metadata
        }
    except Exception as e:
        return {'success': False, 'error': str(e)}, 500
    finally:
        # 清理临时文件
        os.unlink(temp_filename)

if __name__ == '__main__':
    app.run(debug=True)
```

MarkItDown 的转换结果是一个 `DocumentConverterResult` 对象，包含以下主要属性：

| 属性 | 类型 | 描述 |

| `metadata` | dict | 包含标题、作者、创建时间等元数据 |
| `warnings` | list | 转换过程中产生的警告信息 |

### 3. 自动化脚本

```python
#!/usr/bin/env python3
"""自动化文档转换脚本"""

import os
import logging
from pathlib import Path
from markitdown import MarkItDown

# 配置日志
logging.basicConfig(level=logging.INFO, format='%(asctime)s - %(levelname)s - %(message)s')

class DocumentProcessor:
    def __init__(self):
        self.md = MarkItDown()
        self.stats = {
            'processed': 0,
            'successful': 0,
            'failed': 0,
            'errors': []
        }
    
    def process_directory(self, input_dir, output_dir):
        """处理整个目录"""
        input_path = Path(input_dir)
        output_path = Path(output_dir)
        output_path.mkdir(exist_ok=True)
        
        for file_path in input_path.rglob('*'):
            if file_path.is_file():
                self.process_file(file_path, output_path)
        
        self.generate_report()
    
    def process_file(self, file_path, output_path):
        """处理单个文件"""
        try:
            self.stats['processed'] += 1
            
            # 生成输出路径
            relative_path = file_path.relative_to(file_path.parent)
            output_file = output_path / f"{relative_path.stem}.md"
            output_file.parent.mkdir(parents=True, exist_ok=True)
            
            # 执行转换
            result = self.md.convert(str(file_path))
            
            if result.success:
                # 保存结果
                with open(output_file, 'w', encoding='utf-8') as f:
                    f.write(result.markdown)
                
                self.stats['successful'] += 1
                logging.info(f"成功: {file_path} -> {output_file}")
            else:
                self.stats['failed'] += 1
                error_msg = f"转换失败: {file_path}"
                self.stats['errors'].append(error_msg)
                logging.error(error_msg)
                
        except Exception as e:
            self.stats['failed'] += 1
            error_msg = f"异常: {file_path} - {str(e)}"
            self.stats['errors'].append(error_msg)
            logging.error(error_msg)
    
    def generate_report(self):
        """生成处理报告"""
        report = f"""
=== 转换报告 ===
处理文件总数: {self.stats['processed']}
成功: {self.stats['successful']}
失败: {self.stats['failed']}

失败详情:
{'-' * 50}
{'\n'.join(self.stats['errors']) if self.stats['errors'] else '无'}
"""
        print(report)
        
        # 保存到文件
        with open('conversion_report.txt', 'w', encoding='utf-8') as f:
            f.write(report)

if __name__ == "__main__":
    processor = DocumentProcessor()
    processor.process_directory("documents/raw", "documents/processed")
```
